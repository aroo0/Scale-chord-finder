SCALE CHORD FINDER

A program used to find a given chord, e.g. F Minor in minor and major scales. The search engine also finds the function of a chord in a given scale by specifying its number.
The program implements tree and circular list data structure, as well as BFS tree searching algorithm.  Firstly to create scales and chords, and secondly to hold them in the tree structure in order to search for them in an efficient way during conversation.
